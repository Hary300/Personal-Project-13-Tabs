HTML (HyperText Markup Language) digunakan untuk membuat kerangka utama halaman web sebelum ditambahkan gaya dan interaksi. Dengan HTML, kamu bisa membuat elemen seperti heading, paragraf, gambar, dan link.

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan halaman web. Dengan CSS, kamu bisa mengubah warna, font, layout, dan membuat desain menjadi lebih menarik serta responsif di berbagai perangkat.

JavaScript adalah bahasa pemrograman yang membuat website menjadi interaktif. Dengan JavaScript, kamu bisa menangani event, mengubah konten secara dinamis, dan membuat fitur seperti tabs, modal, dan animasi.
